# MiraBlog
## Setup
install dependencies
```
npm install
```
or
```
yarn install
```

### Environment
Create `.env` file with
```
REACT_APP_SERVER_URL=<your_server_endpoint>
```

### Development
```
yarn start
```
for testing
```
yarn test
```

### Production
```
yarn build
```
